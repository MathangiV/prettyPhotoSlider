ALKMaps.Protocol.WeatherAlerts

A basic HTTP protocol for getting weather alerts from the server, and then convert them vector features using WKT reader.  Create a new instance with the ALKMaps.Protocol.WeatherAlerts constructor.

readWeatherAlerts

readWeatherAlerts: function(mapBounds,
alertFilters,
options)

Get weather alerts for a given bounding box.  Returned results can be filtered by urgency, severity and certainty.  This method requests weather alerts first, then requests county polygons.  County polygons are cached for reuse.

Parameters

mapBounds{ALKMaps.Bounds} In degrees.
alertFilters{Object} Like {urgency: “All”, severity: “All”, certainty: “All”}
options{Object} Like {callback: Method, scope: Object }
